Search Help

Typing raw text into the search box will search all properties of every member for matching text. For instance, typing: "woodwork" will show all members whose course list includes the text "woodwork", however it will also include anyone who has the text "woodwork" in any other property, such as their comments.

Searching for a true or false keyword can be done by simply entering the full text of that keyword, any members who have a true value for that keyword (or matching text in another propery) will be shown. Possible keywords are as follows:

'district' 'firstName' 'lastName' 'school' 'email' 'courses' 'emailSecondary' 'aceIt' 'facebook' 'facebookAlias' 'conferences' 'comments' 'psa' 'psaExpires' 'LSA' (LSA/Dist Rep/Pres) 'BCTEA' (exectutive) 'retired' 'retiredYear' 'leave' 'leaveYear' 'academics' 'maker' 'startedTeaching' 'lastUpdated'

Should you wish to perform a search of a specific property, you may use the syntax: property:searchterm. For instance: "facebook:false", or "psaExpires:2016" (dates are treated as text, where a date looks like Mon Sep 28 1998 14:36:22 GMT-0700 (PDT), so you can search for dates by specifying matching text)
